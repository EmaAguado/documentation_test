    Getting Started

    Welcome to the GwaIO Getting Started guide for developers. This section will walk you through the prerequisites you need to integrate GwaIO into your project pipeline, plus tips and examples to help you get up and running quickly.

    Requirements

    Hardware

    • N/A

    Software

    • Python 3 (3.10.10)
    • vscode
    • github

    System Dependencies

    • C/C++ build tools (GCC, Clang)

    OS

    - Windows 10  64 bits (version 21H2) or latest

    Repositories

    Name Git URL Description
    launcher git@github.com:uikokahonia/launcher.git CLI and UI for launching application
    task_schema git@github.com:uikokahonia/task_schema.git Defines and validates task schemas plugin
    publisher git@github.com:mondotv/publisher.git Publishes assets and metadata to storage backends
    syncer git@github.com:mondotv/syncer.git Synchronizes local and server asset repositories
    utilities git@github.com:uikokahonia/utilities.git Utility scripts for DCCs and general pipeline tooling
    jobmanager git@github.com:uikokahonia/jobmanager.git Services for queuing and dispatching render farm jobs
    gwaio_dist git@github.com:mondotv/gwaio_dist.git Distribution executable for gwaio plugins and extensions
    gwaio-docs git@github.com:mondotv/gwaio-docs.git MkDocs Material documentation for the gwaio pipeline
